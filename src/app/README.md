# App Layer

## Overview

Next.js App Routerのルーティング層です。このディレクトリはNext.jsのファイルベースルーティングシステムに従い、URLパスとページコンポーネントの対応を定義します。

## Responsibilities

- **ルーティング定義**: ファイルパスに基づいてURLルートを自動生成
- **メタデータ管理**: 各ページの`metadata`エクスポートでSEO情報やページタイトルを設定
- **レイアウト管理**: `layout.tsx`でアプリケーション全体の共通レイアウトを定義
- **ページエントリーポイント**: 実際のページコンポーネント（`pages/`層）を呼び出す薄いラッパー

## Rules

- この層ではビジネスロジックを含めない
- `pages/`層のコンポーネントを呼び出すだけの薄いラッパーとして実装
- メタデータやルーティング設定のみを担当
- コンポーネントの実装は`pages/`層に委譲
