# Pages Layer

## Overview

ページ単位のコンポーネントを定義する層です。ユーザーが直接アクセスする画面全体を構成します。

## Responsibilities

- **ページ全体の構成**: 1つのURLパスに対応する完全なページコンポーネントを提供
- **ウィジェットの組み合わせ**: 複数の`widgets/`を組み合わせてページを構築
- **ページ固有のロジック**: そのページに特有の状態管理やデータ取得
- **レイアウト調整**: ページ全体のレイアウトやスタイリング

## Rules

- `widgets/`、`features/`、`entities/`を組み合わせてページを構築
- ページ間で共通するロジックは`widgets/`や`features/`に抽出
- 下層（`widgets/`、`features/`、`entities/`、`shared/`）のみを参照可能
- 他の`pages/`層のコンポーネントを参照してはいけない
