# Shared Layer

## Overview

アプリケーション全体で共通して使用される再利用可能なコードを定義する層です。どの層からでも参照可能な、最も基本的な部品を提供します。

## Responsibilities

- **UIコンポーネント**: ボタン、入力フィールド、カードなどの基本的なUI部品
- **ユーティリティ関数**: 日付処理、ローカルストレージ操作などの汎用関数
- **設定・定数**: アプリケーション全体で使用する定数や設定値
- **型定義**: 共通で使用される型定義

## Structure

```
shared/
├── ui/
│   ├── Button.tsx         # ボタンコンポーネント
│   ├── Input.tsx           # 入力フィールドコンポーネント
│   ├── Card.tsx            # カードコンポーネント
│   └── Header.tsx          # ヘッダーコンポーネント
├── lib/
│   ├── date-utils.ts       # 日付ユーティリティ
│   └── local-storage.ts    # ローカルストレージ操作
└── config/
    └── constants.ts        # 定数定義
```

## Rules

- どの層からでも参照可能
- 他の層を参照してはいけない（自己完結的であること）
- フレームワークやライブラリに依存しない純粋な実装を目指す
- 再利用性と汎用性を重視
