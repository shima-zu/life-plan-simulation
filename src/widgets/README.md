# Widgets Layer

## Overview

独立した大きなUIブロックを定義する層です。複数の`features/`や`entities/`を組み合わせて、再利用可能な複合コンポーネントを作成します。

## Responsibilities

- **複合UIブロック**: 複数の機能やエンティティを組み合わせた大きなUI単位
- **フォーム全体の管理**: 複数の入力フィールドやバリデーションを含むフォーム全体
- **データフローの調整**: 複数の`features/`や`entities/`間のデータ連携
- **ビジネスロジックの統合**: 複数の機能を組み合わせた複雑な処理

## Rules

- `features/`、`entities/`、`shared/`を組み合わせてウィジェットを構築
- 他の`widgets/`を参照してもよいが、循環依存に注意
- ページ固有のロジックは`pages/`層に委譲
- 再利用可能な大きなUIブロックとして設計
